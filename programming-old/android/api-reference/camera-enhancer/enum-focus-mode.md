# EnumFocusMode

The enumeration that indicates the focus mode.

```java
public enum EnumFocusMode {
   // Lock the focal length.
   FM_LOCKED(1),
   // Keep continuous auto-focus.
   FM_CONTINUOUS_AUTO(2);
}
```
